# Project Overview

This project involves the development of an immersive Digital Twin of a PEHD bottle production line at the HESTIM Smart Factory Connected (SFC). The system serves a dual purpose:

1. **Pedagogical Simulation Platform:** An offline VR training environment where students and operators can learn complex procedures safely.
2. **Connected Industrial Supervision:** A proof of concept for real-time monitoring, bridging physical machine data with a virtual reality interface.

The integration of these features provides an innovative approach to Industry 4.0 training, combining immersive learning with IoT data streams.
