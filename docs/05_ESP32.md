# ESP32 Data Acquisition

The ESP32 microcontroller is used to interface with the physical machines on the SFC line.

## Functionality
- Reads sensor values from the industrial equipment. Currently, this is validated using the extrusion-blow molding machine's temperature.
- Formats the data into JSON payloads.
- Transmits the payload over the local Wi-Fi network via HTTP POST requests to the FastAPI server.

This lightweight edge device ensures that the core factory equipment does not need to handle complex networking protocols or directly communicate with the VR headset.
