# Educational Scenarios

The VR environment includes structured scenarios to evaluate and improve user skills. 

## The Timed Labeling Task
This is the primary implemented scenario. The user must manually label PEHD bottles following a precise sequence. It consists of three sessions:
1. **Guided Session:** Full audio and visual UI assistance to teach the task.
2. **Autonomous Session:** No guidance; relies on memory to execute the task.
3. **Stress Test:** Introduces unexpected constraints (e.g., missing labels) to test adaptability.

The system tracks execution time and manipulation errors (wrong grabs, out of order steps) to evaluate the learning curve.
