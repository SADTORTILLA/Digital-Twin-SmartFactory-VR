# Roadmap & Future Work

The Digital Twin is an evolving project. Here are the planned next steps:

## High Priority
- **Multiplayer Architecture:** Finalize the Unity Netcode integration for the collaborative factory layout scenario.
- **Sensor Expansion:** Expand the IoT pipeline to capture more machine states, conveyor speeds, and fault alerts from the physical factory.

## Medium Priority
- **Robotic Arm Teaching:** Implement VR scenarios to train users on configuring and managing the robotic arms.
- **Dynamic Production Modification:** Allow the VR user to send commands back to the factory to alter production speed or formats (closing the loop).

## Long Term
- **VR Analytics Dashboard:** Integrate a real-time production KPI dashboard within the VR space.
- **Predictive Maintenance:** Add visual indicators and simulations for predictive maintenance based on historical data.
